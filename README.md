# 🕰️ Digital Clock

A simple and responsive Digital Clock built using **HTML**, **CSS**, and **JavaScript**.

---

## ✨ Features

- Real-time updating clock
- 12-hour / 24-hour format support (optional)
- Fully responsive design
- Minimal and clean UI

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla)
